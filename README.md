# Student Management System

A React application with Redux for managing school students.

## Features
- Add new students
- View list of students
- Edit student information
- Delete students
- Search students by name
- Filter students by grade
- Redux state management

## Getting Started

### Install dependencies
```bash
npm install
```

### Run the application
```bash
npm start
```

The app will open at http://localhost:3000

## Technologies Used
- React 18
- Redux Toolkit
- React-Redux
